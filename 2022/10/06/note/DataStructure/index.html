数据结构-概述

线性结构

常见：数组 队列 链表 栈

存储结构：顺序存储结构 链式存储结构

非线性结构

二维数组 多维数组 广义表 树结构 图结构

数据结构-稀疏数组

稀疏数组一般用于二维数组中存在大量无意义的值

行（row） 列（col） 值（value）
1 m n a
[1] 0 3 33
[2] … … …
[3] … … …

一般第一行数据 代表是一个m行n列的二维表，共有a个位置有值

接下来的每一列代表一个指定行列位置的值

数据结构-队列

有序列表 可以用数组或链表实现

数据结构-栈

栈的三种表达式

  • 前缀（波兰表达式）运算符位于操作数之前 ：+-/*1254
  • 中缀 ：（1+1）-41
  • 后缀（逆波兰表达式）运算符位于操作数之后 ：1234-+*

数据结构-二叉树

数组的存在通过下标访问元素，速度快，链式存储添加删除快

树的存在兼顾二者的优点

满二叉树：满足节点数=2^n-1(n为二叉树的层数) 所有叶子节点在n层

完全二叉树：n层从左到右连续 n-1层从右到左连续

前序遍历：中左右

中序遍历：左中右

后序遍历：左右中

举例 以前序为例

顺序存储

顺序二叉树的特点：

  1. 通常只考虑完全二叉树
  2. 第n个元素的左子节点为 2 * n + 1
  3. 第n个元素的右子节点为 2 * n + 2
  4. 第n个元素的父节点为 （n-1)/2

*线索化二叉树

哈夫曼树

给定n个权值作为n个叶子节点，构造一棵二叉树，若该树的带权路径长度（wpl）达到最小，称为最优二叉树，也称为哈夫曼树

哈夫曼编码

可以用于文件压缩

*补码 反码

二叉树的删除

三种删除情况

  • 删除叶子节点
  • 删除节点下还有一个子节点
  • 删除节点下还有两个子节点

平衡二叉树(AVL)

排序算法 平均时间复杂度

希尔排序 O(n log n)
归并排序 O(n log n)
快速排序 O(n log n)
堆排序 O(n log n)
计数排序 O(n+k)
桶排序 O(n+k)
基数排序 O(n×k)
