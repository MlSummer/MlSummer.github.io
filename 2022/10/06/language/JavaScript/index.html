JavaScript教程

JavaScript是属于HTML和WEB的编程语言

JavaScript 是 web 开发者必学的三种语言之一：

  • HTML 定义网页的内容
  • CSS 规定网页的布局
  • JavaScript 对网页行为进行编程

简介

getElementById() 是多个 JavaScript HTML 方法之一。

使用

在HTML中，JavaScript 代码必须位于 <script> 与 </script> 标签之间。

注释：旧的 JavaScript 也许会使用 type 属性：<script type="text/javascript">

注释：type 属性不是必需的。JavaScript 是 HTML 中的默认脚本语言。

JS 函数和事件

JavaScript函数是一种 JavaScript 代码块，它可以在调用时被执行。

例如，当发生事件时调用函数，比如当用户点击按钮时。

JS 所在位置

可以是<head>头部标签，可以是<body>。

也可以在外部文件，有以下一些优势

  • 分离了 HTML 和代码
  • 使 HTML 和 JavaScript 更易于阅读和维护
  • 已缓存的 JavaScript 文件可加速页面加载

代码块：可以用{...}组合在代码块中定义一同执行的语句

JS 语法

JavaScript 语法是一套规则，它定义了 JavaScript 的语言结构。

JS 标识符

标识符用于命名变量（关键词/函数/标签）

构造变量名称（唯一标识符）的通用规则是：

  • 名称可包含字母、数字、下划线和美元符号
  • 名称必须以字母开头
  • 名称也可以 $ 和 _ 开头（但是在本教程中我们不会这么做）
  • 名称对大小写敏感（y 和 Y 是不同的变量）
  • 保留字（比如 JavaScript 的关键词）无法用作变量名称

**提示：**数值不能作为标识符首字符，大小写区分，小驼峰命名法，使用Unicode字符集

JS 注解

注释：使用单行注释最为常见。

**提示：**注释块常用于官方声明。

可以使用注释来阻止代码执行，适用于测试环节

JS 数字

书写JavaScript数值既可以带着小数点，也可以不带；var x = 3.14;var y = 3

超大或超小的数可以通过科学技术法来写：

JavaScript 数值始终是 64 位的浮点数

与许多其他编程语言不同，JavaScript 不会定义不同类型的数，比如整数、短的、长的、浮点的等等。

JavaScript 数值始终以双精度浮点数来存储，根据国际 IEEE 754 标准。

此格式用 64 位存储数值，其中 0 到 51 存储数字（片段），52 到 62 存储指数，63 位存储符号：

值(aka Fraction/Mantissa) 指数 符号
52 bits(0 - 51) 11 bits (52 - 62) 1 bit (63)

JS DOM

通过这个对象模型，JavaScript 获得创建动态 HTML 的所有力量：

  • JavaScript 能改变页面中的所有 HTML 元素
  • JavaScript 能改变页面中的所有 HTML 属性
  • JavaScript 能改变页面中的所有 CSS 样式
  • JavaScript 能删除已有的 HTML 元素和属性
  • JavaScript 能添加新的 HTML 元素和属性
  • JavaScript 能对页面中所有已有的 HTML 事件作出反应
  • JavaScript 能在页面中创建新的 HTML 事件

HTML DOM 方法是您能够（在 HTML 元素上）执行的动作。

HTML DOM 属性是您能够设置或改变的 HTML 元素的值。
