SQL语句（高级篇）

操作符

TOP/LIMIT/ROWNUM

用于规定要返回的记录的数目

LIKE

用于在WHERE子句中搜索列中的指定模式

IN

允许在WHERE子句中规定多个值，指定列值符合条件中的任意一个

BETWEEN

用于选取介于两个值之间的数据范围内的值(包含边界值)

别名

UNION

用于合并两个或多个SELECT语句的结果集

分组（GROUP BY）

GROUP BY 语句用于结合聚合函数，根据一个或多个列对结果集进行分组。在WHERE操作后

筛选（HAVING）

在 SQL 中增加 HAVING 子句原因是，WHERE 关键字无法与聚合函数一起使用。

HAVING 子句可以让我们筛选分组后的各组数据。

存在判断（EXISTS）

EXISTS 运算符用于判断查询子句是否有记录，如果有一条或多条记录存在返回 True，否则返回 False。

通配符

通配符可用于替代字符串中的任何其他字符（通配符一般LIKE关键字一同使用）

通配符 描述 案例
% 替代0个或多个字符 ‘%a%’
_ 替代一个字符 ‘_a_‘
[charlist] 字符列中的任何单一字符 ‘XDJ’(X.D.J其中一个单词)
[^charlist] 或 [!charlist] 不在字符列中的任何单一字符 ‘！XDJ’(没有X.D.J任意一个)

MySQL中使用REGEXP和NOT REGEXP操作正则表达式

连接（JOIN）

SQL JOIN子句用于把两个或多个表的行为结合起来，基于这些表之间的共同字段。

  • INNER JOIN：如果表中有至少一个匹配，则返回行
  • LEFT JOIN：即使右表中没有匹配，也从左表返回所有的行
  • RIGHT JOIN：即使左表中没有匹配，也从右表返回所有的行
  • FULL JOIN：只要其中一个表中存在匹配，则返回行

函数 描述

CURDATE() 返回当前的日期

DATE() 提取日期或日期/时间表达式的日期部分

DATE_SUB() 从日期减去指定的时间间隔

SQL Date 数据类型

MySQL 使用下列数据类型在数据库中存储日期或日期/时间值：

  • DATE - 格式：YYYY-MM-DD
  • DATETIME - 格式：YYYY-MM-DD HH:MM:SS
  • TIMESTAMP - 格式：YYYY-MM-DD HH:MM:SS
  • YEAR - 格式：YYYY 或 YY

SQL Server 使用下列数据类型在数据库中存储日期或日期/时间值：

  • DATE - 格式：YYYY-MM-DD
  • DATETIME - 格式：YYYY-MM-DD HH:MM:SS
  • SMALLDATETIME - 格式：YYYY-MM-DD HH:MM:SS
  • TIMESTAMP - 格式：唯一的数字

注释：当您在数据库中创建一个新表时，需要为列选择数据类型！

如需了解所有可用的数据类型，请访问我们完整的 数据类型参考手册。

空值（NULL）

如果表中的某个列是可选的，那么我们可以在不向该列添加值的情况下插入新记录或更新已有的记录。这意味着该字段将以 NULL 值保存。

NULL 值的处理方式与其他值不同。

NULL 用作未知的或不适用的值的占位符。

注释无法比较 NULL 和 0；它们是不等价的。

选用NULL和NOT NULL的值

如果字段的内容为NULL，则取值为0的方法

SQL语句（函数）

SQL Aggregate 函数

  1. AVG() - 返回平均值SELECT AVG([字段名]) FROM [表名]
  2. COUNT() - 返回行数，如果选定特定列空值不记录SELECT COUNT(*) FROM [表名];
  3. FIRST() - 返回第一个记录的值SELECT FIRST([字段名]) FROM [表名];
  4. LAST() - 返回最后一个记录的值SELECT LAST([字段名]) FROM [表名];
  5. MAX() - 返回最大值SELECT MAX([字段名]) FROM [表名];
  6. MIN() - 返回最小值SELECT MIN([字段名]) FROM [表名];
  7. SUM() - 返回总和SELECT SUM([字段名]) FROM [表名];
